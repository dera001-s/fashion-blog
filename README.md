Fashion Blog Template
A responsive fashion blog website recreated from the W3Schools Fashion Blog template using Bootstrap 4.

Features: 
Responsive Bootstrap 4 layout
  Hero banner
  Blog post cards
  Sidebar with popular posts, tags, inspiration, and subscription form
 Social media icons
 SEO with Open Graph and Twitter meta tags
  Custom favicon

Technologies Used: 
 HTML5
 CSS3
 Bootstrap 4
 Font Awesome
 Google Fonts
